Neatflixa netflix api demo


I'm sad to annouce the end of Neatflix. It relied on Netflix's public API, which was retired on November 14th, 2014.

I had more plans for Neatflix, but once I realized that Netflix was going to retire their API, it no longer made sense to put more effort into the project.

Thanks to everyone who gave feedback on Neatflix. I hope you enjoyed it! The source code is still available for anyone interested.
